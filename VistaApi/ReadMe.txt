MUST Functional Requirements of the ICA
•	Web Api Services (ThAmCo.Catering) to:
•	Create, edit, delete and list food items 
•	Create, edit, delete and list the details of food Menus 
•	Add and remove a food item from a menu 
•	Book, edit and cancel Food for an Event. The service should return the FoodBookingId as confirmation of the booking;
